## Android Face Recognition

Real Time Face Recognition App using Google MLKit, Tensorflow Lite, & MobileFaceNet.

### Key features

- Detect and Recognize faces in **Real Time**.
- Works **offline** without using API connection.
- **Fast** and **Accurate**.
